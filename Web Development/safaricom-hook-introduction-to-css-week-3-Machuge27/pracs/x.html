1. CSS Basics: Selectors, Properties, and Values

CSS (Cascading Style Sheets) controls how HTML elements appear on a webpage.

‚ú® Structure of a CSS Rule

A CSS rule consists of:

  • Selector: Targets an HTML element
  • Property: Defines what to style
  • Value: Specifies the styling applied
selector { property: value; }

üéØ Example

h1 { color: blue; text-align: center; }

‚ú® Common Selectors

Element Selector

Targets all instances of an element.

Class Selector

Targets elements with a specific class.

Inline, Internal, and External CSS

‚ú® Inline CSS

Define styles directly in an element's style attribute.

<p style="color: red;">This text is red.</p>

‚ú® Internal CSS

Use the <style> tag in the <head> of your HTML document.

<head>   <style>     body { font-family: Arial, sans-serif; }   </style> </head>

‚ú® External CSS

Link an external CSS file using the <link> tag.

<head>   <link rel="stylesheet" href="styles.css"> </head>

üîë Best Practice: Use external CSS for better organization and reusability.
